# git-cheatsheet.md

## Änderungen prüfen

```powershell
git status
```

---

## Alle Änderungen vormerken

```powershell
git add .
```

---

## Commit erstellen

```powershell
git commit -m "beschreibung der Änderung"
```

Beispiele:

```powershell
git commit -m "add home screen"
git commit -m "add game logic"
git commit -m "update settings view"
```

---

## Änderungen zu GitHub hochladen

```powershell
git push
```

---

## Commit-Historie anzeigen

```powershell
git log --oneline
```

---

## Standard-Workflow

```powershell
git status
git add .
git commit -m "beschreibung der Änderung"
git push
```

## Merksatz

```text
status → add → commit → push
```
