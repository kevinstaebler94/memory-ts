# Terminal Cheat Sheet

## Navigation

```powershell
pwd
```

Aktuellen Pfad anzeigen

```powershell
cd Ordnername
```

In Ordner wechseln

```powershell
cd ..
```

Eine Ebene zurück

```powershell
dir
```

Ordnerinhalt anzeigen

---

## Ordner & Dateien

```powershell
mkdir Ordnername
```

Ordner erstellen

```powershell
New-Item datei.ts -ItemType File
```

Datei erstellen

---

## Vite Projekt

```powershell
npm create vite@latest .
```

Vite-Projekt erstellen

```powershell
npm install
```

Dependencies installieren

```powershell
npm run dev
```

Dev-Server starten

---

## VS Code

```powershell
code .
```

Ordner in VS Code öffnen

```powershell
code . -r
```

Aktuelles VS-Code-Fenster verwenden
